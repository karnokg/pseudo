### Requirements

* flex 2.6.4
* bison 3.8.2
* LLVM version

### Building the Source code

1. Checkout Pseudo:
	* `git clone https://github.com/karnokg/pseudo.git`

2. Build using make:
	* `make pseudo`

3. Done!

### Compiling your code written in Pseudo

1. In terminal: 
	* `./pseudo <filename>`
